Además de HTTP, existen otros protocolos que podemos utilizar para transferir ficheros. Esto es especialmente relevante en algunas máquinas en las que no se puede emplear HTTP para la transferencia de binarios.
Especialmente característico es el caso de AUTHY, donde afrontamos una máquina Windows de 32 bits sin powershell habilitado. Esto supone la incapacidad de utilizar powershell para transferir ficheros o crear reverse shells.

En esta clase de casos es especialmente util utilizar otros protocolos para la transferencia de ficheros como es el caso de FTP o SMB.
